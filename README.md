# Stabilarity Hub

Public experiments, tools, and open research projects by Stabilarity Research.

## Open Humanoid — Engineering the Autonomous Robot

A fully open, spec-driven research project to design and simulate a humanoid robot from first principles.

**Research series:** https://hub.stabilarity.com/open-humanoid/

### Structure

- `specs/` — engineering specifications for each subsystem (MASTER_SCHEMA.md)
- `simulation/` — Three.js browser simulation (2 autonomous walking robots)
- `articles/` — companion notes and roadmap for the 20-article series

### Subsystems (20-article arc)
Locomotion · Actuation · Structure · Power · Compute · Vision · Sensor Fusion · Manipulation · Speech · Navigation · Force Control · Safety · RTOS · Multi-Robot Comms · Assembly · Simulation Room
