# Medicaid Open Data Analysis

Analysis of CMS State Drug Utilization Data (SDUD) 2018-2024 from data.medicaid.gov.

## Data Source
- **Dataset:** State Drug Utilization Data (SDUD)
- **Source:** data.medicaid.gov (SODA API)
- **Years:** 2018-2024
- **Records:** ~35M total (sampled via API)
- **License:** Public domain (U.S. government data)

## Charts (13 visualizations)
1. Annual spending trend ($16B→$28B, +71%)
2. Top 20 costliest drugs (HUMIRA dominates)
3. State-level spending distribution
4. Price variance scandal (same drug, 171x-21M x price difference between states)
5. Opioid spending trends (declining from 4.6% to 1.1%)
6. Brand vs generic cost ratio (3,112x unit price premium)
7. State spending growth rates
8. Volume vs cost bubble chart
9. Fraud susceptibility index (composite score)
10. Spending concentration (Pareto/Lorenz — top 20 drugs = 38.3%)
11. Seasonal patterns by quarter
12. Drug price shock — biggest movers
13. State data suppression rates

## Methodology
- Monochrome design (Stabilarity Research Hub palette)
- All data from official CMS/Medicaid government sources
- Fraud index is analytical construct, not official measure
